# Nestle —— 贴贴

Nestle each other!

# 流体

贴贴流体：在流体中的玩家会渴望贴贴

# 物品

贴贴(?)：选择范围内玩家，冲刺过去并增加贴贴值（消耗品）

* 使用 9 个线合成

贴绳(?)：选择玩家贴过去（无耐久）

* 使用贴贴与绳子合成

贴贴雷达：查看选择玩家贴值

* 使用指南针与贴贴合成
* 指向最近的实体

# 药水

贴贴与粗制药水合成  
贴贴药水：效果时间内持续将贴到最近的实体

集合药水使用贴贴药水与发酵蛛眼合成  
集合药水：吸引周围所有实体贴过来

# 分摊伤害

贴贴值到达一定程度的时候，玩家周围可均匀分摊伤害
被分摊的伤害可链条传导下去

# 行动

贴贴：选择玩家并贴过去，要求贴值，和物品是两种反应

# 背景

根据距离，玩家之间会自动添加贴值
